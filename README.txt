McMillen Website (Static)

How to run locally:
- Open index.html in a browser
OR
- Serve with any static server, e.g.
  python -m http.server 8000
  then visit http://localhost:8000

Structure:
- index.html
- styles.css
- app.js
- assets/*.png  (exported from provided PDFs)

Notes:
- Contact form is demo-only. It opens a mailto draft. Replace with your backend endpoint if needed.
- Language toggle: EN / 中文.
